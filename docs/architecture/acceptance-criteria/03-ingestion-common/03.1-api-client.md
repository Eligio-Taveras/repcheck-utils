### New/Extended Classes

#### `CongressGovPaginatedClient[F[_], T]` â€” Trait

Tagless-final base trait parameterized by the raw DTO type `T`. Each entity pipeline provides its own `T` (e.g., `BillListItemDTO`, `AmendmentListItemDTO`) and implements `fetchPage` for its endpoint.

This trait is **Congress.gov JSON API only** â€” paginated endpoints with API key authentication and shared query parameter patterns. Senate.gov / clerk.house.gov XML feeds use the separate `XmlFeedClient` (Â§3.2).

DTOâ†’DO transformation is NOT this trait's responsibility â€” that belongs in the domain object layer (Component 1).

`fetchAll` provides a **concrete pagination loop** (using `tailRecM` for stack safety) that repeatedly calls `fetchPage` until `items.size < pageSize`. Entity clients only need to implement `fetchPage`; they inherit pagination for free â€” matching the existing `PagingApiBase` pattern.

> **`fetchDetail` is NOT part of this trait.** The pagination trait is parameterized by the *list item* DTO type `T`, but detail endpoints return a *different* type (e.g., `MemberDetailDTO` vs `MemberListItemDTO`). Each entity client that needs detail fetching defines its own `fetchDetail` method with the correct return type â€” this avoids a type mismatch that would prevent compilation.

```
trait CongressGovPaginatedClient[F[_], T] {
  def fetchPage(params: FetchParams): F[PagedResponse[T]] =
    Sync[F].raiseError(new UnsupportedOperationException("fetchPage not implemented"))

  /** Concrete â€” paginates by calling fetchPage in a tailRecM loop. */
  def fetchAll(params: FetchParams): Stream[F, T] = { /* pagination loop */ }
}
```

**Fields in `FetchParams`:**
- `congress: Option[Int]` â€” filter by congress number
- `fromDateTime: Option[Instant]` â€” inclusive lower bound
- `toDateTime: Option[Instant]` â€” inclusive upper bound
- `sort: SortOrder` â€” defaults to `UpdateDateDesc`
- `pageSize: Int` â€” defaults to 250 (Congress.gov max)
- `offset: Int` â€” starting offset, defaults to 0

**`PagedResponse[T]`:**
- `items: List[T]` â€” decoded entities for this page
- `totalCount: Int` â€” total available (from pagination.count)
- `nextOffset: Option[Int]` â€” None when no more pages
