### Acceptance Criteria — 1.7 Prompt Chain Base Traits

**Functional:**
1. `PromptStage` enum has values: System, Persona, Lens, Context, Guardrails, Output, Custom. Each has a Circe codec and `fromString` parser.
2. `InstructionBlock` has Circe codecs for reading from GCS (YAML/JSON block files).
3. `PromptProfile` has Circe codecs for reading profile definitions from GCS.
4. `ChainAssembler.assemble` orders stages in the defined pipeline order: System > Persona > Lens > Context > Custom > Guardrails > Output.
5. `ChainAssembler.assemble` applies `WeightTranslator` to each block's content before concatenation.
6. `ChainAssembler.assemble` merges multiple blocks within the same stage (e.g., multiple lenses) into a single stage section.
7. `ChainAssembler.assemble` injects context values (bill text, user preferences) into Context stage blocks by replacing placeholder tokens.
8. `ChainAssembler.assemble` returns `Left` if a referenced block name is not found in the blocks map.
9. `WeightTranslator` applies these rules:
   - 1.0: "You MUST..." strong emphasis wrapping
   - 0.7-0.9: Standard instruction framing
   - 0.3-0.6: "When possible..." / "Consider..." advisory framing
   - < 0.3: Soft suggestion, de-emphasized

**Error Handling:**
10. `ChainAssembler.assemble` returns `Left` with descriptive message listing all missing block names.
11. `InstructionBlock` decoder returns clear error if weight is outside 0.0-1.0 range.

**Test Requirements:**
12. Unit test for `ChainAssembler`: assemble a profile with one block per stage, verify correct ordering.
13. Unit test for `ChainAssembler`: assemble a profile with multiple Lens blocks, verify they appear in the Lens section together.
14. Unit test for `ChainAssembler`: profile references a missing block name, verify `Left` with the block name in the error.
15. Unit test for `ChainAssembler`: context injection replaces `{{bill_text}}` placeholder with actual bill text.
16. Unit test for `WeightTranslator`: weight 1.0 wraps with "You MUST", weight 0.5 wraps with "When possible", weight 0.2 wraps with soft suggestion.
17. Round-trip codec tests for `InstructionBlock` and `PromptProfile`.
